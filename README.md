# Lucy Wachu - Week 2 Portfolio (CSS Mastery)

## Overview
This is my Week 2 portfolio assignment for CSS Mastery. It demonstrates:

- Responsive design
- Flexbox layout
- Grid layout
- Typography system
- Color scheme
- Styled buttons and forms

All pages are styled using a clean, hardcoded CSS file (`styles.css`) without CSS variables for simplicity and compatibility.

## Pages
- **Home (`index.html`)** – Hero section and featured projects
- **About (`about.html`)** – Information about me
- **Projects (`projects.html`)** – Project cards with descriptions
- **Contact (`contacts.html`)** – Contact form

## Features
- Fully responsive across mobile, tablet, and desktop
- Hover and transition effects for buttons and cards
- Accessible form with proper labels and placeholders
- Clean, modern design suitable for portfolio showcase

## Technologies Used
- HTML5
- CSS3 (hardcoded values, no CSS variables)
