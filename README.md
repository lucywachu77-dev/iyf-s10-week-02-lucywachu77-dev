## 📌 Project Description
This is my Week 2 portfolio assignment for CSS Mastery.  

It demonstrates responsive design, clean layouts, and CSS styling principles including Flexbox, Grid, typography systems, and mobile-first responsiveness.

The portfolio includes multiple projects showcasing HTML and CSS skills.

---

## 🛠 Technologies Used
- HTML5  
- CSS3 (no CSS variables used)

---

## ✨ Features
- Fully responsive layout (mobile, tablet, desktop)
- Flexbox and Grid layout for project cards
- Styled buttons with hover effects
- Accessible contact form with labels and placeholders
- Clean and modern UI design
- Multi-page navigation system

---

## ▶️ How to Run
1. Clone this repository  
2. Open `index.html` in your browser  
3. Ensure `styles.css` and `images/` folder are in the correct structure  

---

## 📚 Lessons Learned
- Structuring a multi-page portfolio using HTML & CSS  
- Using Flexbox and Grid for responsive layouts  
- Building a consistent typography and color system  
- Fixing relative path issues for images and navigation  
- Understanding GitHub Pages deployment  

---

## ⚠️ Challenges Faced
- Fixing broken image paths (`../` vs `./`)  
- Resolving GitHub Pages 404 errors  
- Understanding correct file linking between pages  
- Debugging navigation inconsistencies
